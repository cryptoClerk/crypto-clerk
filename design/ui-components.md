# CryptoBooks — Design System

## Brand Identity

**Name:** CryptoBooks
**Tagline:** "Financial docs for crypto freelancers"
**Personality:** Professional but approachable. Not "degen crypto bro" — more "your accountant uses this."

## Components

### Button
- Primary: bg-primary, text-white, rounded-lg, px-4 py-2
- Secondary: bg-transparent, border-primary, text-primary
- Ghost: bg-transparent, text-primary (for subtle actions)

### Card
- bg-white, rounded-xl, shadow-sm, border border-gray-100
- Padding: 24px
- Hover: shadow-md transition

### Input
- bg-white, border border-gray-200, rounded-lg
- Focus: ring-2 ring-accent
- Error: border-red-500, text-red-500 helper

### Receipt Preview
- A4 aspect ratio container
- White background, subtle shadow
- Header: business logo + name
- Body: transaction details in clean table
- Footer: "Generated by CryptoBooks" + timestamp

---

## Layout Patterns

### Landing Page
1. Hero: big headline + tx hash input (try it now)
2. Social proof: "Used by 500+ crypto freelancers"
3. Features: 3 cards (receipts, statements, invoices)
4. Pricing: Free / Pro / Business
5. FAQ
6. Footer

### Dashboard
- Sidebar: Logo, nav (Receipts, Statements, Invoices, Settings)
- Main area: content + actions
- Top bar: user avatar, plan badge, notifications

### Receipt Generator Flow
1. Input page: tx hash + chain (auto-detected)
2. Details form: client, description, business info
3. Preview: live PDF preview
4. Download: PDF + save to account

## Responsive Breakpoints

| Name | Width | Notes |
|------|-------|-------|
| Mobile | < 640px | Single column, stacked inputs |
| Tablet | 640-1024px | Two columns where possible |
| Desktop | > 1024px | Full layout, sidebar nav |

## Accessibility
- Color contrast ratio ≥ 4.5:1 for text
- Focus states visible on all interactive elements
- Semantic HTML (button, not div onclick)
- Alt text on images
- Keyboard navigation works for all flows
