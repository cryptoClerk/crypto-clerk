# CryptoBooks — Competitive Analysis

## Direct Competitors

### CryptoBankStatement
**What:** Basic pay-per-use crypto transaction summaries
**URL:** cryptobankstatement.com

**Strengths:**
- Exists (first mover in niche)
- Simple: paste address, get PDF

**Weaknesses (opportunities for us):**
- No subscription model (pay-per-use = friction)
- No receipt generator
- No invoice builder
- No wallet dashboard
- No API for agents
- Weak branding/customization
- No auto-categorization
- Single chain support

**Our advantage:** Bundle + subscription + agent-ready API

---

## Indirect Competitors

### Koinly
**Focus:** Tax reporting for traders
**Why not us:** Complex, trader-focused, no receipts/statements for income proof

### CoinTracker
**Focus:** Portfolio tracking + taxes
**Why not us:** Same as Koinly — built for investors, not earners

### CoinLedger (formerly CryptoTrader.Tax)
**Focus:** Tax software
**Why not us:** Tax-only, no ongoing income documentation

### Manual Solutions
**What:** Etherscan screenshots + Excel spreadsheets
**Why we win:** Time saved, professionalism, credibility with landlords/lenders

## Defensibility Strategy
1. **Speed to feature breadth** — ship full bundle before they catch up
2. **Agent ecosystem** — API moat is hard to replicate
3. **Community/brand** — "built by freelancers, for freelancers"
4. **Data network effects** — more users → better auto-categorization
