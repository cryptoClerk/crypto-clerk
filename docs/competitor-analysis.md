# Competitor Analysis: Crypto Payment Infrastructure

## 1. CoinTracker / Koinly

**What they do:** Crypto tax calculation and portfolio tracking
**Target:** Individuals filing taxes
**Weaknesses vs CryptoClerks:**
- No invoicing system (can't request payment)
- No receipt generation (just transaction history)
- No payment detection (doesn't know when an invoice is paid)
- Tax-focused, not business operations focused
- No AI agent integration (no MCP, no API for automation)
- Charges per transaction for tax reports

**Our advantage:** We handle the full payment lifecycle: invoice → payment → receipt. They only do tax calculation at the end.

## 2. Request Network

**What they do:** Decentralized payment requests on blockchain
**Target:** Businesses requesting crypto payments
**Weaknesses vs CryptoClerks:**
- Requires smart contract knowledge (complex for average user)
- No automatic receipt generation after payment
- No bank-style statements
- No USD value calculation for historical prices
- Limited chain support
- No AI agent integration (no MCP server)
- No PDF generation for documents

**Our advantage:** We auto-detect payments and generate receipts. They just create payment requests and stop there.

## 3. Loop Crypto

**What they do:** Crypto subscription and recurring payments
**Target:** SaaS companies wanting crypto subscriptions
**Weaknesses vs CryptoClerks:**
- Focused on recurring payments, not one-time invoices
- No receipt generation for individual transactions
- No statements for accounting
- No manual invoice creation for freelancers
- No AI agent access
- Expensive for small businesses

**Our advantage:** We handle any payment type (one-time, recurring, partial). We generate receipts, statements, and invoices. We're built for freelancers and agencies, not just SaaS.

## Summary: Our Unique Position

**CryptoClerks is the only tool that:**
1. Creates invoices with QR codes for crypto payment
2. Auto-detects blockchain payments
3. Auto-generates receipts from detected payments
4. Links receipts to invoices (closes the loop)
5. Generates bank-style statements
6. Provides API + MCP server for AI agents
7. Charges flat monthly fee, not per transaction

**The auto-receipt loop is the moat:** Invoice → Payment → Receipt, all automatic. No competitor does this.
